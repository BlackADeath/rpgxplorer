You can also use rule pages to show maps or other images.

Step 1: Use your favorite map software to create a map OR use an existing map image that you have.

Step 2: Copy the map image into the UserDocs folder in the installation directory.

Step 3: Use Edit Rule Page on the map component and then use your chosen HTML editor to insert the image into the rule page.

RPGXplorer GM Tools will contain a fully-featured map viewer and annotator and will allow you to directly link from a map image to encounters, monsters, npcs, treasure and other components.
